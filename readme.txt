Divulgue ícones com links de suas redes sociais em seu site de forma simples e rápida? Então esse plugin é pra você,em poucos cliques você disponibiliza suas principais redes sociais onde você quiser em seu site, através de um Widget, simples e rápido. Faça o download do plugin WPDEV SOCIAL NETWORKS.
 
== Description ==
Este plugin ativa um widget com campos personalizados, para inserir as principais redes sociais, veja abaixo:
1. Facebook
2. Twitter
3. Youtube
4. Instagran	
 
 
== Installation ==
 
Após você baixar o plugin, deve seguir o processo abaixo:
Ativar o plugin > Acessar a opção de Widgets > Inserir o link da Rede Social, Salvar.
Pronto! Suas redes sociais irão aparecer no local do widget  no seu tema!
